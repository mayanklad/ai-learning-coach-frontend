# AI Learning Coach Frontend

**Description:**  
This repository contains the frontend for the AI-driven Personal Learning Coach. Built with **React** and **Tailwind CSS**, it connects to the backend via **Axios** to display dynamic skills, tasks, and progress charts.

## Features

- Add and manage skills
- Display AI-generated tasks from backend
- Complete tasks and update skill proficiency
- Visualize skill progress with charts
- Responsive and modern UI with Tailwind CSS

## Tech Stack

- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **API Calls:** Axios
